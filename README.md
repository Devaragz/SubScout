# SubScout

SubScout is a web-based dashboard designed to help you effortlessly track and manage your monthly subscriptions. 

## ✨ Features
* **Interactive Dashboard:** View active and expired subscriptions at a glance.
* **Data Visualization:** Includes category breakdown (pie chart) and spending trends (line graph).
* **Responsive UI:** Dark-themed, modern interface built with Bootstrap that works seamlessly on desktop and mobile.
* **Export Options:** Easily download your subscription data as a CSV file.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, Bootstrap 5, Chart.js
* **Backend:** Python (Flask/Django) 
* **Database:** SQLite/PostgreSQL (Update this based on your setup!)

## 🚀 How to Run Locally
1. Clone the repository: `git clone https://github.com/yourusername/subscout.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `python app.py` (Update if your run command is different)
